# SIRH
Programming project for a human resources management application. Project carried out as part of the "IFT785 - Object-Oriented Approaches" course in the Master's programme in computer science at the Universit√© de Sherbrooke.

## Setup

In back-end run :

```mvn spring-boot:run```

In front-end run :

```npm run dev```

The back-end is available on :

```http://localhost:8080/```


The front-end is available on :

```http://localhost:5173/```

API Documentation is available on :

```http://localhost:8080/swagger-ui/index.html```
